## 重要的类的解释（详见注释）
- `PEntity类`：游戏内各种实体（如MC中的玩家、生物、矿车/盔甲架）的基类，用于维护实体各种基本性质

- `PStateMachine`类：有限状态机，管理各种状态

- `PEntityStatsManager`类：管理实体的各类数值

- `PBuffManager`类：管理实体的给雷Buff

- `PAnimator`类：管理实体的各种动画
